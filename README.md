
## Malilang - first somali programming language


### Example Program
```javascript
hawl soo_dhoweey(magac) {
    daabac "Soo dhowey " + magac;
}

weel wey_kujirtaa = haa;

haddii wey_kujirtaa {
    daabac "Tiradu waa sax";
} kale {
    daabac "Tiradu ma aha sax";
}
```
